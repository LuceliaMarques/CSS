Linguagem de estilo CSS

O Cascading Style Sheets (CSS) ou Folha de Estilos em Cascata, em tradução livre, trata-se de uma linguagem de formatação de conteúdo muito utilizada por permitir a modificação do visual (cores, fontes, espaçamento, etc.) de vários elementos ou seções diferentes de uma mesma página de uma só vez. Sua principal função é a de separar os códigos de estruturação e apresentação do conteúdo, tornando-os mais simples de serem escritos e atualizados.

A sintaxe utilizada pelo CSS

seletor { propriedade: valor }
